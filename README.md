# Digital-Cafe-Web-page-basic
Its a web page called Mofu , with an array of dishes you can add and get invoice . it also has a Database . Very useful for new Cafes that need Digital experience and interactions with customers .   
Its easy to use 
first install node_modules in the system , make a folder public . store html files in it. and download the images store them in public folder and make a new folder in public called pics that where u store pics. after that go to mongo db atlas ,  set up a account make cluster and connect using driver . get the link and paste in server.js with i have written to paste . thats it .
